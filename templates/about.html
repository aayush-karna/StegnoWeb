Hi there, Welcome to StegnoWeb. As, the name suggests 'Stegno' stands for Steganography, which is the practice of concealing a file, message, image, or video within another file, message, image, or video. In this website we are demonstrating the basics of steganography i.e, concealing text into image. If you want to know more about Steganography you can go here.

Encoding

Ok, so let me tell you more how encoding works. We ask user for a image and text they want to embed within image. The values of each pixel of image is then taken and converted into binary digits. Since the last digit doesn't have any noticible change it is then replaced with one binary digit of text. The process goes on until each digit of text is embedded into image. And finally a delimeter is added. A delimeter can be anything(for eg: $$$$$), this helps us know that this is the end of text embedded while decoding.

Decoding

As, I said earlier the same process continues while decoding. After user uploads image, the values of each pixel of image is extracted and converted into binary. Then it keeps extracting the last digit of every binary digit and adding it all together until delimeter is found. The delimeter is then extracted and the binary digit is then converted into character.
